# 🧮 Simple Calculator

A clean and user-friendly calculator web application for basic arithmetic operations.

## ✨ Features

- ➕ Addition
- ➖ Subtraction  
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear button to reset
- ⌨️ Keyboard support (optional - agar implement kiya hai to)

## 🎯 Live Demo

👉 **[Click Here to Use the Calculator]  https://amna-2003.github.io/Calculator/**

## 🚀 How to Use

1. Click on the number buttons to enter values
2. Click on operator buttons (+, -, ×, ÷) for calculations
3. Press `=` to see the result
4. Press `C` to clear everything

## 🛠️ Technologies Used

- HTML5
- CSS3 
- JavaScript
